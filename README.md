## Como executar

- Você precisa ter o Python 3 instalado.

```bash
# Clone o repositório
git clone https://github.com/0xAA55-P/solucao-exercicios-logica

# acesse uma pasta e execute um exercício
cd 01Simples
python 001.py
```
